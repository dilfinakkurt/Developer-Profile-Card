# 📱 Yazılımcı Kimlik Kartı (Developer Profile Card)

Bu proje, yazılımcıların kendilerini tanıtabileceği ve projelerini sergileyebileceği **mobil tabanlı bir profil uygulamasıdır**.
Uygulama, React Native kullanılarak geliştirilmiş olup kullanıcı etkileşimini artırmak için **oyunlaştırma (gamification)** mekanikleri içermektedir.

---

## 🎯 Projenin Amacı

Bu uygulamanın amacı:

* Yazılımcı profilini mobil ortamda sunmak
* Kullanıcı etkileşimini artırmak
* Oyunlaştırma teknikleri ile uygulamayı daha eğlenceli hale getirmek
* Modern UI/UX tasarım prensiplerini uygulamak

---

## ✨ Özellikler

### 👤 Profil Kartı

* Yazılımcının **adı, uzmanlık alanı ve seviye bilgisi** gösterilir.
* **"İşe Al"** butonuna basıldığında:

  * `müsaitMi` state değeri `false` olur
  * Buton metni **"Projelerde Çalışıyor"** olarak değişir.

---

### 🧠 Oyunlaştırma Sistemi

Uygulamada kullanıcı etkileşimini artırmak için aşağıdaki oyunlaştırma mekanikleri eklenmiştir:

* ⭐ **XP (Deneyim Puanı) Sistemi**
* 🏆 **Başarımlar (Achievements)**
* 📊 **Seviye Sistemi (Leveling)**
* 🎉 **Kutlama Animasyonları**
* 🏅 **Rozetler**

---

### 📑 Uygulama Sayfaları

#### 🏠 Ana Sayfa

* Yazılımcı profil kartı
* Dinamik buton durumu
* Kullanıcı bilgileri

#### 💻 Projeler Sayfası

* Yapılan projelerin listesi
* Her projeye tıklanınca **+15 XP kazanılır**
* Proje inceleme sistemi

#### 🧩 Yetenekler Sayfası

* Yazılım teknolojileri
* Kullanılan araçlar
* Kullanıcının teknik yetenekleri

#### 📬 İletişim Sayfası

* GitHub
* LinkedIn
* E-posta

İletişim bağlantılarına tıklandığında:

* **+10 XP kazanılır**
* Networking başarımına katkı sağlar
